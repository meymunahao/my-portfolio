Sententia

Sententia is a minimalist web application designed to deliver curated, actionable advice with a single interaction. Leveraging an external REST API, the application fetches and renders randomized sayings in real-time. Built with a focus on clean UI/UX principles, the project demonstrates proficiency in asynchronous JavaScript (fetching data), DOM manipulation, and responsive design. It serves as a digital companion for users seeking a moment of clarity or a fresh perspective during their day.

Tech Stack: HTML5, CSS3, JavaScript, API
