A Digital Companion for Cognitive & Emotional Wellness

The Affirmation Cards Description Page is a responsive, content-managed web platform built for The Neurobide. It serves as a digital instructional guide that teaches users how to effectively use physical affirmation card decks through psychological principles and structured routines.

View Live Project

Problem & Solution

The Problem

Although customers loved the physical affirmation cards, many were unsure how to integrate them into their daily routines. Common questions included “What do I do after drawing a card?” and “How does this actually help my brain?” The client needed a centralized, educational resource that could evolve over time.

The Solution

I designed and developed a dynamic knowledge hub using WordPress. Instead of relying on static pages, I built a content management system that allows the Neurobide team to update guides, add scientific explanations, and manage FAQs without developer assistance. The interface was intentionally minimal and calm, supporting a focused learning experience.

Key Features

  • Custom Guide Post Type: A specialized backend structure for adding affirmation methods and steps separately from blog content.
  • Dynamic FAQ Accordion: A JavaScript-powered Q&A section that allows the client to manage frequently asked questions through the WordPress dashboard.
  • Step-by-Step Carousel: An interactive slider that walks users through the four-step affirmation process: Read, Internalize, Speak, and Repeat.
  • Responsive Typography: Carefully scaled text to ensure readability across mobile, tablet, and desktop devices.
  • Video Resource Integration: Support for embedding tutorial videos directly within the instructional pages.

Technologies Used

  • CMS: WordPress (Custom Theme Development)
  • Backend: PHP (Custom loops and post type registration)
  • Database: MySQL
  • Frontend: HTML5, CSS3 (Sass), JavaScript (Vanilla)
  • Tools & Plugins: Advanced Custom Fields (ACF)

What I Learned

  • CMS Customization: I learned how to move beyond default WordPress blogging features to build a fully customized content management experience.
  • User Education UX: This project taught me the importance of instructional design—structuring information so users can learn and apply it effectively.
  • Advanced Custom Fields: I gained hands-on experience using ACF to give clients control over content while maintaining layout consistency.
  • Client Empathy: I learned to design systems that empower clients to manage their own platforms independently, reducing long-term reliance on developers.
